수정 규칙

1. 수정 일시, 내용, 사람 명시
2. PM 제외 Git push 금지
3. Git commit은 수정 할 때마다 필수로
4. 변수명에서의 띄어쓰기는 대문자로 표기(ex: javaScript)
5. Git add는 자신이 수정한 파일만 add
